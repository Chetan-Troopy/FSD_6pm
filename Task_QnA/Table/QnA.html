HTML Table Tag

10 Interview Questions and Answers

# Question Answer
1 What is the <table> tag used for? It defines a table in HTML.
2 What are the main elements of a table? <tr>, <th>, <td>, <thead>, <tbody>, <tfoot>, <caption>.
3 Difference between <th> and <td>? <th> is bold and center-aligned, <td> is normal text.
4 What is colspan and rowspan? Colspan merges columns, Rowspan merges rows.
5 How do you add a border to a table? Use the border="1" attribute.
6 Purpose of <thead>, <tbody>, <tfoot>? They organize table sections.
7 What is the default alignment of text in <th>? Center-aligned.
8 How to create a table with a caption? Use the <caption> tag.
9 How to create a fixed-width table? Use the width attribute.
10 How to add spacing in a table? Use cellspacing and cellpadding.

10 Practical Tasks

# Task Example Code
1 Create a simple table with 3 rows and 3 columns. <table border="1">...</table>
2 Merge two columns using colspan. <th colspan="2">Merged</th>
3 Merge two rows using rowspan. <td rowspan="2">Merged</td>
4 Create a table with <caption>. <caption>Student Table</caption>
5 Create a table with header, body, and footer. <thead> <tbody> <tfoot>
6 Create an attendance table with checkboxes. <input type="checkbox">
7 Create a table with numbered rows. Use numbers in <td>
8 Use a table to display a timetable. Rows for time slots, columns for days.
9 Create a pricing table with 3 product plans. Plan 1 | Plan 2 | Plan 3
10 Create a student report card using a table. Subjects | Marks | Grade
