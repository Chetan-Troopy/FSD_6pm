HTML Interview Questions and Answers

  1. What is the purpose of the <img> tag?
    The <img> tag is used to embed images in a webpage. It requires a src attribute to specify the image URL.
  2. How do you provide alternative text for an image?
    Use the alt attribute inside the <img> tag. This helps with accessibility and SEO.
  3. What is the use of the <a> tag?
    The <a> tag defines a hyperlink, allowing navigation to another page or section.
  4. How do you open a link in a new tab?
    Use the target="_blank" attribute in the <a> tag.
  5. What is an absolute vs. relative URL in an <a> tag?
    An absolute URL contains the full address (e.g., https://example.com), while a relative URL is relative to the current page.
  6. How do you create an anchor link within the same page?
    Use an id on the target element and link to it using <a href="#section-id">.
  7. Can an image be a hyperlink?
    Yes, by placing the <img> tag inside an <a> tag.
  8. How do you create an email link?
    Use <a href="mailto:email@example.com"> to create an email link.
  9. What is the purpose of the rel attribute in <a>?
    It defines the relationship between the linked page and the current page, useful for SEO and security.
  10. How can you make a list of links?
    Use an unordered (<ul>) or ordered list (<ol>) and place <a> tags inside <li> items.
  11. What is the difference between <ul> and <ol>?
    <ul> creates an unordered list, while <ol> creates an ordered list with numbers.
  12. How do you create a navigation bar using lists?
    Use a <ul> with <li> elements, placing links inside each list item.
  13. Can an <img> tag be used as a list item?
    Yes, images can be placed inside <li> elements.
  14. How do you make a responsive image?
    Set width="100%" in CSS or use the max-width property.
  15. What is the purpose of the title attribute in <a>?
    It provides additional information when hovering over a link.
  16. How do you create an image gallery using lists?
    Use a <ul> or <ol> with each <li> containing an image.
  17. What happens if the src attribute of an <img> is missing?
    The image will not load, and the alt text (if provided) will be displayed.
  18. How do you style a navigation menu created using lists?
    Use CSS to remove bullets, display items inline, and add padding.
  19. How do you create a button-like link?
    Use CSS to style an <a> tag as a button.
