HTML Form Tag

10 Interview Questions and Answers

# Question Answer
1 What is the <form> tag used for? It defines an HTML form for user input.
2 What are the common form elements? <input>, <textarea>, <button>, <select>, <label>, <fieldset>.
3 What is the purpose of the action attribute? It specifies where the form data should be sent.
4 What does the method attribute do? It defines how the form data is submitted (GET or POST).
5 Difference between GET and POST method? GET appends data to URL, POST sends data in the request body.
6 What is the purpose of the name attribute in form elements? It assigns a name to an input field for data processing.
7 What is the placeholder attribute? It provides a hint inside the input field.
8 What is the required attribute? It ensures that a field must be filled before submission.
9 What is the difference between <button> and <input type="submit">? <button> can have inner HTML, while <input type="submit"> cannot.
10 How do you create a dropdown list in a form? Using the <select> and <option> tags.

10 Practical Tasks

# Task Example Code
1 Create a simple login form with username and password fields. <input type="text" name="username">
2 Add a submit button to a form. <input type="submit" value="Submit">
3 Create a form with radio buttons for gender selection. <input type="radio" name="gender" value="male"> Male
4 Create a form with checkboxes for multiple selections. <input type="checkbox" name="option1"> Option 1
5 Create a dropdown list for selecting a country. <select> <option>USA</option> </select>
6 Create a form with a file upload option. <input type="file" name="fileUpload">
7 Use a textarea for user feedback. <textarea></textarea>
8 Make an email input field with validation. <input type="email" required>
9 Add a reset button to clear form fields. <input type="reset">
10 Create a form with a hidden input field. <input type="hidden" name="token" value="12345">
