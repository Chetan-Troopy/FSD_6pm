CSS

CSS is used to beautify our HTML document.

CSS is used make Our website Attractive.

Ways To Apply CSS

  • Inline CSS
  • Internal CSS
  • External CSS

Syntax of CSS

  1. Inline CSS Syntax
    • Inline CSS is written by using style attribute
      • style="property1: value1; property2: value2;"
  2. Internal CSS Syntax
    • Internal CSS is written by using style tag in head section.
      •             <style>
                selector{
                    property1: value1;
                    property2: value2;
                }
             </style>
  3. External CSS Syntax
    • External CSS is written by using link tag in head section.
      • 
            <link rel="stylesheet" href="style.css">
