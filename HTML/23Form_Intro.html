Form In HTML

action => Backend file path

method => We have two Methods

  1. get => by default, limitation of character, not a secure method, shows data in the url, not file supported
  2. post => No limitation, secure method, Never shows data in the url, file supported

enctype

  • application/x-www-form-urlencoded => By Default
  • multipart/form-data => Method must be Post,
  • text/plain => For Debuuging Purpose
